# EVENTIA Frontend

Application Next.js pour:
- espace organisateur (dashboard)
- experience invitation invite (token securise)

## Script
- `npm.cmd run dev`
- `npm.cmd run build`
- `npm.cmd run start`

## Variable d'environnement
- `NEXT_PUBLIC_API_URL=http://localhost:4000`

## Pages principales
- `/`
- `/organisateur`
- `/organisateur/login`
- `/organisateur/register`
- `/organisateur/dashboard`
- `/invitation/[token]`
